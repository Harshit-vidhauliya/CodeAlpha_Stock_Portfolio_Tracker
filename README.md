# CodeAlpha Stock Portfolio Tracker

This is my Stock Portfolio Tracker project developed during the CodeAlpha Python Programming Internship.

# About
This project calculates the total value of a stock portfolio using predefined stock prices. The user enters the stock name and quantity, and the program displays the total investment value.

# Features
- User input for stock name and quantity
- Predefined stock prices
- Calculates total investment
- Simple and easy to understand

- Python
